# 本地 AI 流量哨兵 (AI Network Sentinel)

> 基于本地大模型的 Web 流量威胁分析系统 | 毕设项目

![架构图](docs/architecture.png)

## 📌 项目简介

本项目利用 **LangChain + Ollama** 在 **NVIDIA RTX 4060** 笔记本端本地部署 **Qwen2.5-7B** 大语言模型，结合 Kali Linux 的 TShark 抓包能力，实现对 HTTP 流量的语义级安全分析。系统能够识别 SQL 注入、XSS 等常见 Web 攻击，并输出**中文解释性报告**，全程数据不出本地，保障隐私安全。

## ✨ 核心功能

- **流量采集**：通过 TShark 捕获 HTTP 请求并提取关键字段
- **AI 语义分析**：调用本地大模型识别攻击行为并生成中文报告
- **可视化仪表盘**：Streamlit + ECharts 展示风险等级分布
- **一键启动**：提供批处理脚本，双击即可运行

## 🛠️ 技术栈

- **AI 框架**：LangChain、Ollama
- **大模型**：Qwen2.5:7B (4-bit 量化)
- **前端**：Streamlit、ECharts
- **抓包工具**：TShark (Kali Linux)
- **硬件加速**：NVIDIA RTX 4060 Laptop GPU + CUDA 12.7
- **语言**：Python 3.11

## 🚀 快速启动

### 环境要求
- Windows 11 / Windows 10
- NVIDIA 显卡 (显存 ≥ 8GB)，已安装 CUDA 驱动
- Python 3.11
- Ollama (需提前下载 Qwen2.5:7B 模型)

### 安装步骤

1. 克隆仓库
\`\`\`bash
git clone https://github.com/你的用户名/AI_Network_Sentinel.git
cd AI_Network_Sentinel
\`\`\`

2. 创建并激活虚拟环境
\`\`\`bash
python -m venv venv
venv\Scripts\activate   # Windows
\`\`\`

3. 安装依赖
\`\`\`bash
pip install -r requirements.txt
\`\`\`

4. 启动 Ollama 服务（需提前下载模型）
\`\`\`bash
ollama run qwen2.5:7b
\`\`\`

5. 运行 Streamlit 应用
\`\`\`bash
streamlit run src/app.py
\`\`\`

或者双击项目根目录下的 `start_ai_sentinel.bat` 一键启动。
